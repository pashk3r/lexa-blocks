# Lexa Blocks

Образовательная игра на Python, разработанная в команде в рамках учебного проекта. Игра является аналогом Block Blast, но с образовательным элементом: вместо просмотра рекламы для продолжения игры после проигрыша нужно ответить на вопросы, сгенерированные нейросетью.

## Технологии

- **Python**
- **Pygame** - игровой движок и рендеринг
- **DeepSeek API** - генерация вопросов
- **Git**

## Структура разработки

Проект велся по этапам, отражённым в ветках репозитория:

- `analysis` - анализ требований и планирование
- `design` - проектирование архитектуры и игровых механик
- `development` - непосредственная реализация функционала
- `testing` - тестирование и отладка

## Установка на Windows

1. **[Скачать игру (Windows)](https://github.com/pashk3r/lexa_blocks/raw/development/LexaBlocks.exe)**
2. Запустите скачанный файл `LexaBlocks.exe`
3. Играйте!
