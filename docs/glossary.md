# Глоссарий игровых сущностей — Lexa-Blocks

**Версия:** 3.0 (20.05.2026)

---

| Сущность | Атрибут | Тип данных | Описание |
|---|---|---|---|
| **Поле** | Ширина | Целое число = 8 | Количество ячеек по горизонтали |
| | Высота | Целое число = 8 | Количество ячеек по вертикали |
| | Сетка | Двумерный массив | Хранит состояние каждой ячейки |
| **Ячейка** | Занята | Логический (да/нет) | Занята ли ячейка фигурой |
| **Фигура** | Форма | Список координат | Относительные координаты кубов, из которых состоит фигура |
| | Цвет | Цвет | Цвет для отображения на экране |
| **Спасательный куб** | Форма | Одиночная клетка | Частный случай фигуры размером 1×1 |
| **Очки** | Текущий счёт | Целое число | Сумма всех заработанных очков |
| **Уровень** | Номер уровня | Целое число | Вычисляется как целая часть от (счёт / 100) + 1 |
| **Прогресс** | Процент | Целое число (0–100) | Остаток от деления счёта на 100. 1 очко = 1% |
| **Обучающий вопрос (Python)** | Текст вопроса | Строка | Формулировка вопроса по Python |
| | Тип вопроса | Перечисление | `choice`, `input`, `match` |
| | Варианты ответа (`choice`) | Список строк | Четыре варианта: `A) ...`, `B) ...`, `C) ...`, `D) ...` |
| | Правильный индекс (`choice`) | Целое число (0–3) | Индекс правильного варианта в списке |
| | Правильный ответ (`input`) | Строка | Точный ответ: слово, число или короткое выражение (строчными буквами) |
| | Левый столбец (`match`) | Список строк | `["a) ...", "b) ...", "c) ...", "d) ..."]` |
| | Правый столбец (`match`) | Список строк | `["1) ...", "2) ...", "3) ...", "4) ..."]` |
| | Соответствие (`match`) | Строка | Строка в формате `"a3b1c4d2"`, где буква = термин, цифра = определение |
| **Состояние игры** | Текущее состояние | Перечисление | `PLAYING`, `LOSS_MENU`, `WAITING_QUESTION` |
| **Набор фигур в панели** | Количество | Целое число = 3 | Три фигуры, доступные игроку в текущий момент |
| **Запас кубов** | Количество | Целое число | Накопленные спасательные кубы, без ограничения |
