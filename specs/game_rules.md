# Игровые правила Lexa-Blocks

## Поле
- Размер 8×8 ячеек.

## Фигуры
- 5 типов: квадрат 2×2, линия 3×1 вертикальная, линия 3×1 горизонтальная, угол L, одиночный куб.
- В панели всегда 3 случайные фигуры. Новый набор появляется только когда все три размещены.

## Очки и прогресс
- За каждый куб размещённой фигуры игрок получает 1 очко.
- За каждую полностью заполненную строку или столбец (8 ячеек) игрок получает 10 очков.
- Если за один ход очищено несколько линий, очки суммируются.
- Прогресс = текущий счёт % 100 (0–99). Отображается как процент до следующего уровня.
- Уровень = floor(счёт / 100) + 1. Каждые 100 очков повышают уровень.

## Спасательные кубы
- За правильный ответ на математический вопрос игрок получает 3 куба размером 1×1.
- Кубы можно использовать после успешного ответа на вопрос.
- Лимита на количество кубов нет.

## Условие тупика (Game Over)
- После каждого успешного размещения фигуры проверяются все 3 фигуры в панели.
- Если ни одну из них нельзя разместить, игра переходит в состояние LOSS_MENU.

## Спасение через математический вопрос
- Доступно в LOSS_MENU.
- Вопрос генерируется внешним API, возвращает 4 варианта ответа (A, B, C, D).
- Таймаута на ответ нет.
- Кнопка «Сдаться» засчитывается как неправильный ответ.
- При правильном ответе: добавляются 3 куба, игра продолжается (поле и счёт не меняются).
- При неправильном ответе или сдаче: полный сброс (счёт=0, поле очищено, кубы=0, уровень=1).
- При ошибке соединения с API показывается диалог с возможностью повторить или сдаться.
